To identify the type of hash, we could use a tool such as `hashid`, `hash-identifier`, or `hashcat.

**Hash-Identifier**
```
hash-identifier
```
Then, paste the hash.

**HashID**
```
# For Single Hash
hashid -m hash.txt

# For multiple hashes:
hashid -f hashes.txt
```
