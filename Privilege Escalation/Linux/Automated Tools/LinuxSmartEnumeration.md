**Linux Smart Enumeration (LSE)** is another shell script that enumerates information of a system for ***potential privilege escalation path***.

**Run the basic script**
```bash
./lse.sh
```

#### Expanding the output 
**Level 1 verbosity**
```bash
./lse.sh -l1
```

**Level 2 verbosity**
```bash
./lse.sh -l2
```
